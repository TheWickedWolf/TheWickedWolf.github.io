About

Medium: Blender & Substance Painter

Genre: Game Design

Description

This was my first 3-D object that I designed in my Game Design class. It is based on a medieval anvil and took me 3 days to complete. Since this was my first attempt at making a game object, I followed a tutorial to help me learn the features and tools of Blender. The full mesh and UV coordinates were done in Blender while I textured it in Substance Painter. I tried to give it a well-worn look to make it look older. Overall, I worked very hard on this to get it viable for a game engine to use.
